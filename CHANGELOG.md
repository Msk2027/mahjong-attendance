# 変更履歴 (Changelog)

このファイルには、麻雀出欠ボードの主な変更をまとめています。
形式は [Keep a Changelog](https://keepachangelog.com/ja/1.0.0/) を参考にしています。

## [1.1.0] - 2026-06-03

### ✨ 新機能 (Added)
- **ルーム自発退出**：メンバー（owner以外）が自分でルームから退出できるようにしました。

### 🔧 改善・整理 (Changed)
- **メンバー操作を詳細モーダルに集約**：メンバーをタップすると詳細が開き、そこから昇格/降格・退会ができるようにしました（スマホでのボタン位置のガタつきを解消）。
- **メンバー・招待を専用ページに分離**：ルーム画面から「詳細」ページ（`/room/[roomId]/detail`）を新設し、メンバー一覧・招待をそちらへ移動。ルーム画面は日程・出欠まわりに集中できるよう整理しました。
- **ルーム削除 / 退出を詳細ページのカードに集約**：ルーム削除（オーナーのみ）・退出（メンバー）は詳細ページのカードから行う形に変更しました。
- **ルーム画面のレイアウト整理**：入れ子になっていた「ゲスト追加」「開催確定」を独立セクションに分け、候補日カードも縦並びに整理しました。
- **ホーム画面の整理**：参加中ルーム一覧を、タップで開けるシンプルな行に変更（招待コード/URL・ロール表示などを整理）。ログイン中の表示からメールアドレスを省き、設定ボタンをログアウトの隣に移動しました。
- **見た目の統一**：各ページの「戻る」やルーム画面のボタン類をボタン型に統一し、使われていなかった Room ID 表示を削除しました。

## [1.0.0] - 2026-06-03

### ✨ 新機能 (Added)
- **ルーム削除**：オーナーのみ、ホーム画面とルーム画面からルームを削除できるようにしました（関連データもまとめて削除）。
- **Adminロール**：`owner` / `member` に加えて `admin` を追加。オーナーが任命でき、開催確定・候補削除・確定取消などの運営操作が可能です（「ルーム削除」と「メンバーのロール変更」はオーナー限定）。
- **メンバー強制退会**：オーナーは admin / member を、admin は member を、ルームから退会させられます（member は不可）。
- **パスワード再設定**：ログイン画面に「パスワードを忘れた方はこちら」を追加し、メールから再設定できる専用ページ `/reset-password` を新設しました。

### 🐛 修正 (Fixed)
- ログイン画面で「新規登録 ↔ ログイン」を切り替えた際に、ユーザー名の入力欄（と値）が残ってしまう不具合を修正しました。
- ログイン画面のタイトルが、モードに応じて「ログイン」/「新規登録」と切り替わるようにしました。

### 🔧 その他 (Changed)
- パスワード再設定ページをダークテーマに合わせて読みやすく調整し、リンクからのセッション検出を頑丈にしました。
- ルーム運営系の操作（開催確定・各種削除・ロール変更・強制退会）の権限チェックを、データベース側（RPC）でも行うように強化しました。
